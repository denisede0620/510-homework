510 作業管理系統 V2 使用說明

1. 解壓縮 ZIP。
2. 將 index.html、style.css、script.js 上傳到 GitHub Repository 根目錄。
3. GitHub → Settings → Pages。
4. Source 選 Deploy from a branch。
5. Branch 選 main，資料夾選 /(root)，按 Save。
6. 等候約 1–3 分鐘後開啟 GitHub Pages 網址。
7. 第一次使用掃描時，允許瀏覽器使用相機。

學生 QR Code 內容必須和試算表「學生名單」B欄 QR編號完全相同，例如 51001。

Apps Script 部署需設定：
- 執行身分：我
- 存取權：任何人
