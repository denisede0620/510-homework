# 作業小幫手

GitHub Pages 前端，資料寫入 Google Apps Script 與 Google 試算表。

## 上傳到 GitHub
1. 進入 `denisede0620/510-homework`
2. 點「新增檔案」→「上傳檔案」
3. 上傳 `index.html`、`style.css`、`script.js`
4. 按「提交變更」

## 開啟 GitHub Pages
1. 儲存庫「設定」
2. 左側「頁面」
3. 來源選「從分支部署」
4. 分支選 `main`，資料夾選 `/(root)`
5. 儲存

網站網址：
https://denisede0620.github.io/510-homework/

## Apps Script 部署必要設定
- 執行身分：我
- 存取權：任何人
- 修改 Code.gs 後必須建立新版本並重新部署
